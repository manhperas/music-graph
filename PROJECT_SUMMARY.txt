PROJECT OVERVIEW
-------------------------------------------------------------------------------
A complete Python-based graph network analysis system that:
1. Scrapes 1000+ US-UK pop artists from Vietnamese Wikipedia
2. Extracts and parses infobox data (genres, instruments, albums, etc.)
3. Builds a collaboration graph network
4. Imports to Neo4j graph database
5. Computes network statistics and creates visualizations

USAGE EXAMPLES
-------------------------------------------------------------------------------

1. COMPLETE PIPELINE:
   $ ./run.sh all

2. INDIVIDUAL STAGES:
   $ ./run.sh collect    # Scrape Wikipedia
   $ ./run.sh process    # Clean data
   $ ./run.sh build      # Build graph
   $ ./run.sh import     # Import to Neo4j
   $ ./run.sh analyze    # Statistics & visualization

3. NEO4J QUERIES:
   # Count nodes
   MATCH (n) RETURN labels(n), count(n)
   
   # Top artists
   MATCH (a:Artist)-[r]-()
   WITH a, count(r) AS degree
   ORDER BY degree DESC LIMIT 10
   RETURN a.name, degree
   
   # Find collaborations
   MATCH (a1:Artist)-[:PERFORMS_ON]->(album)<-[:PERFORMS_ON]-(a2:Artist)
   RETURN a1.name, album.title, a2.name LIMIT 20
